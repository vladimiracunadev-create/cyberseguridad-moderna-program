# 🕸️ Web — El JWT indiscreto ⭐

Una app te entregó este token de sesión (JWT). El desarrollador cree que, como está
"firmado", su contenido es secreto. Demuéstrale que no.

```text
eyJhbGciOiAiSFMyNTYiLCAidHlwIjogIkpXVCJ9.eyJ1c2VyIjogImd1ZXN0IiwgInJvbGUiOiAiYWRtaW4iLCAiZmxhZyI6ICJGTEFHe2p3dF9wYXlsb2FkX25vX2NpZnJhZG99In0.c2lnbmF0dXJlZmFrZQ
```

**Pista:** un JWT son tres partes separadas por puntos, codificadas en Base64URL. El
*payload* es la del medio.
